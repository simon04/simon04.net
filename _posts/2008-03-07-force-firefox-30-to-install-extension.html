--- layout: post title: Force Firefox 3.0 to install Extension tags: - misc status: publish type: post published: true meta: _edit_last: '1' --- Hello, there are several older add-ons hanging around there which don't support Firefox 3.0 officially. Therefore Firefox quits the installation on 2 reasons:
  1. ... could not be installed because it is not compatible with Firefox ...
  2. ... will not be installed because it does not provide secure updates.
To fix these problems add a new boolean value in the about:config's table and set the value to be false each.
  1. extensions.checkCompatibility
  2. extensions.checkUpdateSecurityÂ
